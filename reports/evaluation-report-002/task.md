# 评估任务

## 任务描述
使用 TeaQL Agent Kit，用 Rust 构建一个学校管理系统。

## 领域概念
- Platform（平台，域根）
- School（学校）
- School Type（学校类型，常量：Primary, Secondary）

## 任务要求
1. 首先创建语义 TeaQL 模型
2. 评估模型
3. 生成 Rust TeaQL 代码
4. 构建查询 API 和注册 API
5. 使用生成的 Q 查询 API 和 E 表达式
6. 不手动编辑生成的 TeaQL 服务代码

## 任务边界
- 所有文件放在 ~/workspace/A-003 文件夹
- 遵循 TeaQL Agent Kit 的 autonomous 分支规范
- 遵循 AGENTS.md 中的所有规则
